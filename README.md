<h1 align="center">FoxDebloater 🦊🚫</h1>

<p align="center">
  Egy <code>.reg</code> fájl, ami kitakarítja a Firefoxodból a felesleges dolgokat.
</p>

---

## ✨ Mit csinál?

| | |
|---|---|
| 📊 Telemetria | Adatgyűjtés, crash report, health report — ki |
| 🧹 Bloat | Pocket, Screenshots, ajánlók — viszlát |
| 🔒 Privacy | DNS-over-HTTPS, Tracking Protection — be |
| ⚡ Speed | Kevesebb háttérfolyamat = gyorsabb Firefox |

---

## 🚀 Használat

1. `FoxDebloater.reg` → Jobb klikk → Futtatás rendszergazdaként
2. Firefox újraindít 💜

---

## 🔄 Visszaállítás

`FoxRestore.reg` → Mindent visszaállít. Profil, jelszavak, könyvjelzők maradnak.

---

## 🛠 Ellenőrzés

`about:policies` a címsorba → "Managed by your organization" = működik

---

## ⚠️ Fontos

A Firefox hivatalos policy rendszerét használja. Semmi hack. Ha gond van, ott a restore.
